Cara Dapatkan API Key YouTube

Ikuti langkah berikut (gratis, 30 detik selesai):

  1. Buka rapidapi.com/hub dan daftar (pakai Google bisa)
  2. Di search bar, ketik: "YouTube Video Download"
  3. Pilih API "YouTube Video Downloader" oleh judith12 (atau yang serupa)
  4. Klik "Subscribe to Test" (pilih paket Basic - gratis 50 request/bulan)
  5. Setelah subscribe, di halaman API ada kode "X-RapidAPI-Key"
  6. Salin key tersebut dan tempel di kolom API Key di atas

Key disimpan di browser kamu saja (localStorage). TikTok tetap berfungsi tanpa API key.
